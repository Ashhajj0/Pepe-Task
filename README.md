# PepeTask - Telegram Mini App

A premium, mobile-first Telegram Mini App themed after Pepe, built with React, Vite, Tailwind CSS, and Firebase.

## Features
- **Telegram SDK Integration**: Automatically detects user identity and profile photo.
- **Firebase Firestore**: Auto-syncs users on login.
- **Sophisticated Dark UI**: Custom-crafted interface optimized for the Telegram environment.
- **Real-time Balance**: Tracks user PEPE balance (initialized at 0).

## Tech Stack
- Frontend: React 19 + Vite
- Styling: Tailwind CSS 4
- Icons: Lucide React
- Animations: Motion (formerly Framer Motion)
- Backend: Firebase Firestore

## Deployment to Vercel
1. Push this code to a GitHub repository.
2. Connect the repository to Vercel.
3. Vercel will automatically detect the Vite build settings.
4. Deploy!

## Telegram Setup
1. Open [@BotFather](https://t.me/botfather).
2. Create a new bot or select an existing one.
3. Set the "Menu Button" or create a "Web App" link pointing to your Vercel URL.
